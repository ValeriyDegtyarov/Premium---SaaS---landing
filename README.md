Учебный проект — полный лендинг по Figma макету (webocean.io).

## О проекте

Верстал самостоятельно в режиме реального заказа: получал макет, 
сдавал секции заказчику, получал правки. Без готовых шаблонов и 
Bootstrap — только чистый HTML и CSS.

## Стек

- HTML5 (семантическая разметка)
- CSS3 (Flexbox, Grid, анимации, адаптив)
- CSS Custom Properties
- BEM-именование
- Vanilla JS (бургер-меню, sticky хидер, карусель отзывов)

## Что реализовано

- 9 секций по Figma макету
- Адаптив 768px
- Бургер-меню на мобайл
- Sticky хидер при скролле
- Карусель отзывов
- Hover эффекты на кнопки и карточки
- CSS архитектура: отдельный файл на каждую секцию
